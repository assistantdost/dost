It does not just answer questions.

Ask naturally, and DOST can take action across your desktop and cloud tools.

From "open my work apps" to "check my calendar and send an email,"

## Default Server Setup (Simple and Powerful)

DOST comes with two default MCP servers.

### 1. Desktop Server (Local Stuff)

This is your local action engine.
It handles on-device workflows and system-level tasks.

Best for:

- launching and managing apps
- window and system controls
- notes, reminders, screenshot, clipboard
- local productivity automation

### 2. Remote Server (Cloud Stuff)

This is your online services engine.
It handles integrations and live data.

Best for:

- Google (Gmail, Calendar, Contacts)
- Spotify playback and search
- weather, stock, crypto, metals, currency

In short:

- desktop server = local actions
- remote server = cloud integrations

## Works with Any MCP Client

DOST default servers are MCP-standard.

That means they are not limited to one app.
They can be used by any MCP-compatible client.

You can run them with:

- the DOST desktop client
- the DOST CLI client
- custom internal MCP clients
- other third-party MCP clients
